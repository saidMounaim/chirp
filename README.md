# Chirp

![Chirp](https://i.ibb.co/C7gg5XT/chirp-bg.png)

Chirp is a platform for short and sweet posts built using Next.js 13, Prisma, and Tailwind CSS. It allows users to create and publish posts, as well as comment on posts created by other users. It's a simple and minimalist app that's perfect for those who want to share their thoughts in a quick and easy way.

## Getting Started

To get started with Chirp, follow these steps:

1. Clone this repository: `git clone https://github.com/saidMounaim/chirp.git`
2. Install the dependencies: `npm install`
3. Set up the database: `npx prisma migrate dev`
4. Run the development server: `npm run dev`

The app should now be running on `http://localhost:3000`.

## Features

Chirp has the following features:

- User authentication: Users can sign up and log in to create and publish posts and comment on posts.
- Create posts: Users can create posts.
- View posts: Users can view all the posts on the platform.
- Comment on posts: Users can comment on posts created by other users.
- Delete posts: Users can delete their own posts.

## Technologies Used

Chirp was built using the following technologies:

- [Next.js 13](https://nextjs.org/)
- [React Query](https://tanstack.com/query/latest)
- [Prisma](https://www.prisma.io/)
- [Tailwind CSS](https://tailwindcss.com/)
